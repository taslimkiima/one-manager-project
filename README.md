One Manager - Sistem Manajemen Keanggotaan (Proyek Manajemen Proyek)
--------------------------------------------------------

Proyek ini adalah simulasi dari aplikasi **One Manager**, sebuah sistem manajemen keanggotaan berbasis web yang dirancang untuk memudahkan organisasi dalam mengelola anggota, grup, dan event. 

**Deskripsi:**
- Sistem ini mengimplementasikan manajemen anggota dengan struktur grup bertingkat (parent-child).
- Memiliki fitur CRUD untuk anggota, grup, dan event.
- Menggunakan Role-Based Access Control (RBAC) untuk kontrol akses berbasis peran.
- Menyediakan integrasi kalender untuk pengelolaan event dan absensi.
- Fitur dashboard untuk menampilkan ringkasan aktivitas dan statistik anggota.
- Notifikasi email/in-app untuk undangan dan pengingat event.
- Aplikasi ini dibuat menggunakan **metode manajemen proyek Agile**, dengan penekanan pada kolaborasi tim dan iterasi cepat.

**Penting:**
- Ini adalah **folder simulasi** yang digunakan untuk **pemahaman manajemen proyek** dalam konteks pengembangan perangkat lunak. Semua pengaturan dan kode di dalam folder ini **bertujuan untuk mendemonstrasikan** penerapan konsep-konsep manajemen proyek seperti **WBS**, **Gantt Chart**, **Risk Register**, dan lainnya.
  
Repositori ini digunakan untuk menggambarkan bagaimana manajemen proyek diterapkan dalam konteks pengembangan sistem perangkat lunak.

**Catatan:**  
Proyek ini **fiktif** dan digunakan sebagai bahan pembelajaran untuk memahami siklus hidup proyek perangkat lunak serta pengelolaan tim dalam pengembangan perangkat lunak.
